/api/user/register post

post

Registra un usuario en el sistema.

post /api/user/register

Registra un usuario en el sistema.

  • Request
  • Response

Body

Media type: application/x-www-form-urlencoded

Type: object

Properties
  • tipo_persona: required (one of natural, juridica)
  • email: required (string)
  • password: required (string)
  • nombre: (string)

    Requerido para Persona Natural

  • apellido: (string)

    Requerido para Persona Natural

  • cedula: (string)

    Requerido para Persona Natural

  • pasaporte: (string)

    Requerido para Persona Natural

  • nombre_sociedad: (string)

    Requerido para Persona Juridica

  • ruc: (string)

    Requerido para Persona Juridica

HTTP status code 200

Body

Media type: application/json

Type: object

Examples:

success:

{
  "ok": true,
  "user": {
    "tipo_persona": "natural",
    "email": "fulano@example.com",
    "password": "F00lano",
    "nombre": "Fulano",
    "apellido": "De Tal",
    "cedula": "8-123-456",
    "pasaporte": null,
    "nombre_sociedad": null,
    "ruc": null
  }
}

error:

{
  "ok": false,
  "errors": [
    {
      "status": 400,
      "title": "Tipo de persona invalido."
    }
  ]
}
